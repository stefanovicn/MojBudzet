# MojBudzet

Web aplikacija za upravljanje ličnim budžetom razvijena korišćenjem tehnologija PHP, HTML, CSS, JavaScript i SQL, organizovana kroz MVC strukturu.

## Technologies
- PHP
- HTML
- CSS
- JavaScript
- SQL

## Features
- registracija korisnika
- prijava korisnika
- upravljanje kategorijama troškova
- evidencija troškova
